# Kuo Tracker (Flutter)

Мобильное Flutter-приложение для Android/iOS на основе исходного `index.html` трекера.

## Что реализовано

- календарь с отметками дней;
- логирование по напиткам и объёму (мл/л/порции);
- расчёт метрики КУО за месяц;
- рекорд за всё время;
- трезвая серия и базовая статистика;
- локальный лидерборд на `SharedPreferences`.

## Запуск

1. Установите Flutter SDK.
2. Выполните:

```bash
flutter pub get
flutter run
```

> Для генерации нативных папок (`android/`, `ios/`) при необходимости:

```bash
flutter create .
```
